# God · 上帝造人（人物深度蒸馏引擎）

> 上帝造人，赋予的是灵魂。God Skill 蒸馏人，提炼的也是灵魂——
> 不是他们说了什么，而是他们为什么需要成为他们自己。

---

## 核心哲学

大多数人物分析停在表层：**这个人怎么思考 / 怎么表演 / 怎么竞技？**

God Skill 多问一层：**这个人是什么？**

这不是修辞的升级，是蒸馏深度的跃迁。

无论是思想家还是歌手，企业家还是运动员，网红还是演员——
一个人的行为模式背后，总有更深的存在处境在驱动：

- 芒格的跨学科思维，背后是对"被愚弄"的深层恐惧
- 科比的"曼巴精神"，背后是对平庸的存在性恐惧
- 泰勒·斯威夫特的叙事驱动，背后是对"被误解"的持续对抗
- 乔布斯的完美主义，背后是对死亡的不妥协
- 李小龙的截拳道哲学，背后是对"形式囚笼"的突破执念

**行为模式是表层。驱动行为模式的存在处境，才是核心。**

God Skill 提炼七个维度，分属七个层次——**对所有人物类型普适**：

| 层次 | 维度 | 核心问题 |
|------|------|---------|
| 认知层 | 心智模型 | 这个人用什么框架看世界？框架背后是什么驱动力？ |
| 决策层 | 决策启发式 | 这个人如何做判断？什么时候这些规则会失效？ |
| 表达层 | 表达 DNA | 这个人如何输出自己？（语言/作品/身体/表演）情绪如何影响输出？ |
| 价值层 | 价值观层级 | 这个人相信什么？坚持这些信念付出了什么代价？ |
| 谱系层 | 影响谱系 | 这个人从哪里来？在与谁对话或对抗？ |
| 边界层 | 认知盲区 | 这个人看不见什么？盲区从哪里来？ |
| **灵魂层** | **存在处境** | **伤口 · 执念 · 与死亡的关系 · 意义押注 · 核心矛盾** |

## Phase 0：确认目标 + 人物类型判断（60秒）

收到名字后，先做两件事：

### 0.1 基本确认

1. **对象是谁**：确保理解正确，区分同名人物
2. **素材来源**：用户是否提供本地素材？（书稿、访谈稿、歌词、比赛视频转录、私信记录等）
3. **蒸馏深度**：全像蒸馏 vs. 专项维度？是否需要灵魂层？
4. **用途**：思维顾问 / 创作参考 / 深度理解 / 写作素材

> 如果用户直接说"蒸馏X"，不追问 → 默认全像蒸馏含灵魂层，直接开始。

### 0.2 人物类型判断（决定 Agent 配置）

这是 God Skill 的关键判断步骤。不同类型人物，核心信息藏在完全不同的地方：

| 类型 | 核心信息所在 | 激活的专属 Agent |
|------|------------|----------------|
| **思想家 / 学者 / 作家 / 哲学家** | 著作、论文、长访谈 | A1 著作深挖 |
| **企业家 / 投资人 / 管理者** | 决策记录、股东信、产品选择 | A2 商业决策 |
| **音乐人 / 歌手 / 词曲人 / 制作人** | 歌词、专辑叙事弧、现场表演、制作选择 | A3 音乐作品 |
| **演员 / 导演 / 编剧 / 制片人** | 角色选择模式、导演手法、片场行为、剧本批注 | A4 影视作品 |
| **运动员 / 教练 / 武术家** | 训练哲学、比赛关键时刻、伤病应对、退役处理 | A5 竞技表现 |
| **视觉艺术家 / 设计师 / 建筑师 / 摄影师** | 作品系列演变、创作手记、材质/风格选择 | A6 视觉作品 |
| **网红 / 内容创作者 / 播客主 / 直播主** | 选题逻辑、粉丝关系、变现方式、人设演变 | A7 内容创作 |
| **政治家 / 社会活动家 / 革命者 / 宗教领袖** | 演讲、行动记录、盟友/敌手关系、危机处理 | A8 公共行动 |
| **科学家 / 发明家 / 工程师** | 研究方法、实验记录、学术争论、科普方式 | A9 科研创造 |
| **厨师 / 料理人 / 手工艺人 / 工匠** | 技术哲学、食材/材料选择、师承关系、作品演变 | A10 匠艺实践 |
| **教育者 / 老师 / 导师** | 教学方法、学生关系、课程设计、教育理念 | A11 教育传授 |
| **复合型人物** | 多个维度并行 | 多个专属 Agent 组合 |

> **上述类型是参考，不是边界。** 遇到未覆盖的行业（外交官、棋手、游戏设计师、法官、宇航员……），遵循同一原则自主判断：**这个人的核心信息藏在哪里？** 激活信息密度最高的专属 Agent，或启用 AX 自定义 Agent。

**类型判断原则**：
- 一个人可以跨多种类型（如 Jay-Z = 音乐人+企业家+文化符号）
- 优先激活信息密度最高的专属 Agent
- 通用 Agent 对所有类型都运行，专属 Agent 按需叠加
- 遇到全新领域，参考最接近的专属 Agent 逻辑自主扩展

## Phase 0.5：素材策略 + 创建目录

### 素材来源策略

**模式 A：纯网络调研**（无本地素材）
→ 根据人物类型，启动通用 Agent + 对应专属 Agent 并行执行

**模式 B：本地素材优先**（用户提供了文件）
→ 先读本地素材，按维度分类（一份材料可能覆盖多个维度）
→ 识别信息缺口：哪些维度缺失或薄弱？
→ 只对缺失维度启动网络搜索补充
→ 调研文件中明确区分「来自本地素材」vs「来自网络搜索」

**本地素材优先级**：用户提供的一手素材优先级最高，远超网络二手转述。本地已覆盖的维度跳过网络搜索。

## Phase 1：多源调研（并行 Agent 群）

### 通用 Agent（对所有人物类型都运行）

---

#### G1 · 长对话 Agent

**目标**：播客、长视频访谈、AMA、深度对话；本地访谈稿优先

**关键提取**：
- 被逼到墙角时的回应（压力下的真实反应）
- 即兴类比与比喻（揭示底层思维框架）
- 改变过的立场（思想的演变轨迹）
- 拒绝回答的问题（防御区就是敏感区）
- 对话风格：主导还是回应？攻击性还是包容性？

**输出**：`references/research/通用Agent输出/G1-conversations.md`

---

#### G2 · 表达 DNA Agent

**目标**：社交媒体（Twitter/X、微博、Instagram、抖音）、短视频、公开演讲片段、本地短篇素材

**关键提取**：
- 高频词汇与句式（语言指纹）
- 争议立场与公开争论（边界感）
- 幽默风格：讽刺/自嘲/荒诞/冷幽默/无
- 自我叙事模式：如何介绍自己？如何定义成功？
- 情绪表达习惯：愤怒时、兴奋时、脆弱时各怎么说话？

**输出**：`references/research/通用Agent输出/G2-expression-dna.md`

---

#### G3 · 外部视角 Agent

**目标**：他人分析、媒体报道、传记、批评文章、同行评价、粉丝/反粉分析

**关键提取**：
- 外部观察到的行为模式（往往比本人叙述更客观）
- 主要批评与争议（批评揭示边界）
- 同行如何评价？竞争对手如何描述？
- 随时间演变的公众形象与真实形象的落差
- 崇拜者看到的和批评者看到的，各是什么？

**输出**：`references/research/通用Agent输出/G3-external-views.md`

---

#### G4 · 时间线 Agent

**目标**：出生/出道至今的完整时间线；**最近12个月**重点（防止过时）

**关键提取**：
- 人生关键节点与转折（每次转折背后的处境）
- 思想/风格/立场的演变轨迹
- 重大危机与应对方式（危机暴露真实性格）
- 最近动态（新作品、新立场、新争议）
- 哪些节点改变了他们是谁？

**输出**：`references/research/通用Agent输出/G4-timeline.md`

---

#### G5 · 灵魂层 Agent

**目标**：不做新的信息检索，而是综合 G1-G4 全部材料 + 所有专属 Agent 材料，进行**心理考古**

**这个 Agent 最后运行，等所有其他 Agent 完成后启动。**

**伤口线索**：
- 哪些记忆/经历他们反复提及？（重复叙述 = 身份锚点）
- 早年的羞耻/失败/被否定如何塑造了后来的执念？
- 他们在什么话题/场景下防御性升高？（防御区 = 伤口所在）
- 哪些成就永远"不够"？（补偿模式）

**执念线索**：
- 即使没有外部压力，他们会一直做什么？
- 什么是他们的"最后一道防线"——跨越它他们会不再是自己？
- 他们最害怕成为哪种人？

**死亡关系**：
- 他们如何谈论遗产、意义、时间？
- 工作节奏的急迫感从哪里来？
- 他们如何对待同辈的消逝（死亡/退出/失败）？

**意义押注**：
- 他们把存在感押注在什么上？
- 如果这个押注失败，他们还是他们自己吗？
- 他们的工作是在解决什么，还是在逃避什么？

**核心矛盾**：
- 公开声称相信 X，但行为模式显示他们其实相信 Y
- 最强的地方和最弱的地方是否来自同一个源头？
- 试图调和但始终无法调和的是什么？

**灵魂层的所有推断必须有素材支撑，标注依据，不凭空捏造"深刻感"。**

**输出**：`references/research/通用Agent输出/G5-soul-layer.md`

### 专属 Agent（按人物类型激活，可组合）

---

#### A1 · 著作深挖 Agent
**激活条件**：思想家、学者、作家、哲学家

**目标**：书籍、论文、长文、newsletter、演讲全文；本地书稿优先

**关键提取**：
- 重复出现≥3次的核心主张（重复 = 真实信念）
- 自创术语与概念（命名一件事 = 相信它重要）
- 论证结构：归纳还是演绎？类比还是数据？
- 书单与引用习惯（谁影响了他们的思维？）
- 同一主题在不同时期的处理方式（思想演变）

**输出**：`references/research/专属Agent输出/A1-writings.md`

---

#### A2 · 商业决策 Agent
**激活条件**：企业家、投资人、管理者、创业者

**目标**：股东信、产品发布记录、并购/放弃决策、公司危机处理、投资组合逻辑

**关键提取**：
- 反直觉的决策及其逻辑（最能揭示真实框架）
- 资源分配模式：把钱和时间押注在什么上？
- 说"不"的模式：放弃了什么？为什么？
- 危机决策：在最大压力下做了什么？
- 言行一致性：说要做和实际做的差距

**输出**：`references/research/专属Agent输出/A2-business-decisions.md`

---

#### A3 · 音乐作品 Agent
**激活条件**：音乐人、歌手、词曲作者、制作人、乐队

**目标**：歌词文本（逐字分析）、专辑叙事弧、制作/编曲选择、现场表演风格、音乐访谈；本地歌词优先

**关键提取**：
- **歌词主题演变**：从早期到近期，关注点如何变化？（主题变化 = 人生阶段）
- **反复出现的意象与隐喻**（意象 = 内心地图）
- **制作风格选择**：极简还是繁复？自制还是找人？合作选择揭示什么？
- **现场表演能量**：舞台上与台下判若两人还是一致？
- **争议/禁歌/被审查的内容**（被压制的往往最真实）
- **专辑作为整体的叙事结构**：这张专辑想说什么故事？
- **写给谁**：自我疗愈、对话公众、记录历史，还是创造幻象？

**对音乐人特别重要**：歌词是他们最诚实的语言，往往比访谈更真实。

**输出**：`references/research/专属Agent输出/A3-music-works.md`

---

#### A4 · 影视作品 Agent
**激活条件**：演员、导演、编剧、制片人

**目标**：角色选择模式、导演手法分析、片场行为记录、剧本批注（如有）、评论访谈；本地剧本/手记优先

**关键提取**：
- **角色选择模式**（演员）：接了什么拒了什么？是否反复扮演相似人物？
- **导演手法特征**（导演）：镜头语言、叙事节奏、演员调度、对白风格
- **片场行为**：与演员/剧组的相处模式（权威型/协作型/混乱型）
- **商业 vs 艺术的选择记录**：何时妥协，何时坚持？
- **作品中反复出现的主题**：死亡/救赎/背叛/家庭——背后是什么？
- **失败作品的处理方式**（往往比成功作品更能揭示性格）

**输出**：`references/research/专属Agent输出/A4-screen-works.md`

---

#### A5 · 竞技表现 Agent
**激活条件**：运动员、教练、竞技相关人物

**目标**：训练哲学、比赛关键时刻分析、伤病记录与应对、赛后发言、退役/转型处理；本地训练日记/战术手记优先

**关键提取**：
- **训练哲学**：投入的时间量和质，对训练本身的态度（苦行还是热爱）
- **压力下的表现模式**：决定性时刻如何行动？
- **失败应对方式**：输了之后做什么？说什么？恢复多快？
- **伤病关系**：如何对待身体的极限？是否会忽视疼痛？代价是什么？
- **与队友/对手的关系模式**：合作还是个人英雄主义？
- **退役/转型**：何时放手？如何定义运动员身份之外的自己？
- **比赛之外的行动**：社会议题、商业选择、公众发言

**对运动员特别重要**：身体是他们最诚实的语言，训练选择揭示世界观。

**输出**：`references/research/专属Agent输出/A5-athletic-performance.md`

---

#### A6 · 视觉作品 Agent
**激活条件**：视觉艺术家、设计师、摄影师、建筑师

**目标**：作品系列演变分析、创作手记、材质/媒介/风格选择、展览叙事、艺术家自述；本地手稿/草图优先

**关键提取**：
- **作品系列的演变轨迹**：风格如何转变？转变的触发点是什么？
- **材质与媒介的选择逻辑**：为什么用这种材料？背后的执念是什么？
- **反复出现的视觉母题**：形状、颜色、空间关系——内心地图的外化
- **创作过程**：冲动型还是计划型？合作还是孤独工作？
- **商业委托 vs 自主创作**：如何平衡？何时拒绝？
- **对待批评的方式**：防御？吸收？忽视？

**输出**：`references/research/专属Agent输出/A6-visual-works.md`

---

#### A7 · 内容创作 Agent
**激活条件**：网红、YouTuber、播客主、直播主、内容创作者

**目标**：内容选题逻辑、粉丝关系管理、变现路径选择、人设演变轨迹、平台策略；本地运营记录优先

**关键提取**：
- **选题逻辑**：什么话题反复出现？是真实兴趣还是流量策略？

- **人设演变**：早期到现在，呈现的自己如何改变？是成长还是漂移？

**对网红特别重要**：他们的"真实感"往往是最精心设计的产品。

### 每个 Agent 的硬性要求

- 结果必须存入对应 md 文件，没有文件的调研等于没有发生
- 注明来源和可信度（本地一手 > 网络一手 > 二手 > 推断）
- 明确标注来源类型：「本地素材」/「网络搜索」/「推断」
- 区分"本人说的" vs "别人说他们的" vs "我推断的"
- **遇到矛盾，保留矛盾——不要抹平**，矛盾是最有价值的信号
- 灵魂层（G5）的推断必须有素材支撑，标注依据

### 来源优先级

| 来源类型 | 揭示什么 | 权重 |
|---------|---------|------|
| 用户提供的本地一手素材 | 最可靠的原始材料 | **最高** |
| 本人著作/歌词/剧本/作品 | 经过选择的自我表达 | 最高 |
| 长对话/访谈（本地原文） | 即兴思维过程 | **最高** |
| 实际决策/行动记录 | 真实行为 vs 宣称 | 高 |
| 作品演变轨迹 | 价值观的时间线 | 高 |
| 反常行为记录 | 深层动机 | 高 |
| 长对话/访谈（网络） | 即兴思维过程 | 高 |
| 社交媒体/短内容 | 表达风格、即时反应 | 中 |
| 他人评价/批评 | 外部视角、盲区 | 中 |
| 二手叙述 | 参考但需核实 | 低 |

### 来源黑名单（永远排除）

中文人物排除：知乎、微信公众号、百度百科/知道。只接受权威来源：36氪、极客公园、晚点、财新、第一财经、虎嗅、少数派等。

### Agent 失败处理

- **超时/无结果**：不等待，继续。在 Phase 2 标注"信息不足"
- **某个专属 Agent 材料稀缺**（<5条可用来源）：提醒用户，标注置信度低，如有本地素材则使用
- **结果相互矛盾**：保留矛盾——矛盾是宝贵信号，不是需要消除的错误

> **铁律**：一个诚实的60分 Skill（有明确标注的局限）好过一个虚假的90分 Skill（捏造信息填补空白）。

### 矛盾仲裁规则

当不同 Agent 对同一事件/特征有相反描述时，按以下规则处理：

**第一步：判断矛盾类型**

| 矛盾类型 | 处理方式 |
|---------|---------|
| **来源层级不同**（一手 vs 二手）| 高层级来源优先，低层级标注为"存疑" |
| **时间段不同**（早期 vs 近期）| 两者都保留，标注时间段，视为"演变" |
| **语境不同**（公开表态 vs 实际行动）| 两者都保留，行为优先于言辞 |
| **真正的矛盾**（同一语境下相反描述）| 都保留，在 SKILL 中明确呈现为"内在张力" |

**第二步：在 Phase 2 中处理**
- 来源层级矛盾 → 在注释中标注，低可信度来源降权而非删除
- 时间演变 → 在影响谱系或时间线维度中呈现变化轨迹
- 行为/言辞落差 → 直接作为认知盲区或内在张力的素材
- 真正矛盾 → 作为"核心矛盾"写入价值层或灵魂层，不强行消解

**不要为了让 Skill 看起来"一致"而删除矛盾。真实的人都是矛盾的。**

用户确认质量 → 进入 Phase 2。
用户希望某维度补充 → 补充后继续。

## Phase 2：框架提炼（综合合成）

**注意**：提炼框架对所有人物类型普适，但具体内容因类型而异。
音乐人的"表达DNA"重点在歌词；运动员的重点在身体语言和赛场行为；网红的重点在选题和人设。
**用人物类型匹配的语言和维度来填写，不要强行套用思想家的框架去描述运动员。**

#### 2.1 心智模型提炼（认知层，3-7个）

识别标准：
- **跨域重复出现**：同一框架在2+个不同情境出现 → 真实心智模型
- **有自创术语/概念**：命名了这种思维模式 → 核心信念
- **有预测力**：能预测他对新情境的反应

每个模型记录：
- 名称与一句话定义
- 来源证据（≥2个场景，注明来源类型）
- 运用方式与触发条件
- **局限与失效条件**（必须有，这是诚实的标志）
- **驱动力追问**：是什么存在处境让他需要这个模型？

#### 2.2 决策启发式提炼（决策层，5-10条）

= 这个人做判断时使用的快速规则。格式："如果X，那么Y"，有具体案例支撑。

每条记录：
- 启发式规则
- 来源证据（具体决策/行动案例）
- **适用边界**：什么情况下会产生反效果？
- **情绪触发点**：哪些场景会让理性规则被情绪覆盖？

#### 2.3 表达 DNA 分析（表达层）

**根据人物类型选择对应的表达维度：**

| 人物类型 | 表达 DNA 的核心维度 |
|---------|-------------------|
| 思想家/作家 | 论证结构、词汇精度、类比习惯、确定性表达 |
| 音乐人/歌手 | 歌词意象、韵律偏好、主题演变、声音质地 |
| 演员/导演 | 角色诠释方式、肢体语言、镜头语言、沉默的使用 |
| 运动员 | 身体语言、赛场决策风格、胜负时的反应、赛后语言 |
| 网红/创作者 | 选题风格、人设一致性、与观众的距离感、情绪调用 |
| 艺术家/设计师 | 视觉语言、材质选择、作品密度、留白习惯 |
| 科学家/工程师 | 论证严密度、对不确定性的表达、科普隐喻偏好 |
| 厨师/工匠 | 对食材/材料的描述方式、对"完美"的语言、师承叙事 |
| 教育者 | 提问方式、类比习惯、对学生错误的反应、耐心/急躁的边界 |
| 其他行业 | 参照最接近的类型，聚焦「专业行为」中的语言和选择模式 |

通用维度（所有类型都提炼）：
- 情绪状态与输出的关系：愤怒时...；脆弱时...；巅峰时...
- 确定性表达："我不确定"型 vs "显然"型
- 幽默风格：讽刺/自嘲/荒诞/冷幽默/无

#### 2.4 价值观与反模式（价值层）

- **价值观**：3-5个排序后的核心价值观
- **价值观的代价**：坚持这些价值观让他们失去了什么？
- **反模式**：这个人明确反对或本能回避的行为/状态
- **矛盾与张力**：价值观之间的内在冲突（深度的来源）

#### 2.5 影响谱系（谱系层）

- 谁影响了他们 → 他们 → 他们影响了谁
- 在什么谱系/传统中工作？（音乐流派/运动员谱系/思想学派）
- **对话对象**：他们在作品或行动中与谁在争论？

**如何从素材中提炼"对话对象"**：找他们反复引用、反驳、或刻意区别于的人物/思想。他们在访谈中说"我不同意X的观点"或"我想做和Y不一样的事"，这些就是他们真正的对话对象——有时候比正式的师承更能揭示思想立场。

#### 2.6 认知盲区（边界层）

- 他们系统性看不见什么？
- **盲区的心理起源**：是补偿机制还是认知局限？
- 在什么类型的情境下最容易产生偏差或失误？

#### 2.7 灵魂层提炼（灵魂层）

从 G5 的调研中结构化提炼四个切面：

**2.7.1 伤口图谱**
- 核心伤口：最深的、反复被触碰的心理创伤
- 补偿模式：伤口如何转化为动力或执念？
- 触发点：什么场景会让这个人"回到"伤口状态？
- 置信度标注

**2.7.2 执念结构**
- 核心执念：他们不能停止的事
- 执念的边界：跨越什么他们会不再是自己？
- 执念的代价：为此付出了什么？

**2.7.3 与死亡的关系**
- 他们如何谈论遗产、意义、时间？
- 工作节奏揭示的急迫感从哪里来？
- 他们的创作/竞技/行动是在建造什么，抵抗什么消逝？

**2.7.4 意义押注**
- 他们把存在感押注在什么上？
- 如果这个押注失败，他们还是他们自己吗？
- 这个押注是真实的信念，还是对恐惧的防御？

#### 2.8 诚实边界

必须明确声明：
- 无法预测对全新情境的反应
- 无法替代此人的直觉、才能与创造力
- 公开表达与私下想法可能不同
- 信息截止于调研日期
- **灵魂层是推断，不是事实**——基于公开素材的解读，可能是错的
- **所有人都比任何框架更复杂**——这个 Skill 是工具，不是真相

# 查理·芒格 · 认知操作系统

## 此 Skill 能做什么，不能做什么

**能做**：用芒格的跨学科框架分析投资、商业和人性问题；识别自己思维中的系统性偏误；

## 核心心智模型（认知层）

### 1. 多元思维模型格栅
**一句话**：用100+个跨学科模型构建一张格栅，任何问题在格栅上都会被多角度照亮。
**运用方式**：遇到一个问题，先问"哪些学科有处理这类问题的基本原理"，从物理、生物、心理学、经济学等多个角度分别分析，再综合。
**触发条件**：任何需要做重要判断的情境，尤其是投资决策。
**局限**：在高度专业化的技术判断上效率不如领域专家；在需要快速决策的情境下，格栅思维耗时太长。
**驱动力**：他对"单一视角被证明是错的"有一种近乎偏执的恐惧——格栅是防止被单一学科欺骗的护甲。

### 2. 逆向推理（反过来想）
**一句话**：要想知道如何成功，先彻底想清楚如何失败，然后避开那些路。
**运用方式**：对任何目标，先问"什么会导致这件事彻底失败？"列出所有失败路径，逐一排除，剩下的才是值得做的事。
**触发条件**：评估一个新项目、投资机会或生活决策时。
**局限**：逆向思维擅长排除，但不擅长在多个同样可行的选项中挑选最优；对于需要创造性突破的问题效果有限。
**驱动力**：他更信任"避免愚蠢"而不是"追求聪明"。这来自他的基本人性观：聪明是稀缺的，愚蠢是普遍的。

### 3. 人类误判心理学
**一句话**：人类有25种系统性心理倾向会导致错误判断，了解它们才能不被自己和他人愚弄。
**运用方式**：在做决策前，主动检查自己是否陷入激励偏误、社会认同偏误、过度自信等陷阱；在评估他人行为时，先问"是什么心理机制在驱动这个行为"。
**触发条件**：遇到任何看起来"太好了不像真的"的机会，或者发现自己或他人行为异常时。
**局限**：这个框架更擅长解释行为而非预测行为；在极度非理性的市场环境中，知道偏误存在但无法预测它何时纠正。
**驱动力**：他认为大多数人生悲剧来自可预防的愚蠢，而不是无法抗拒的命运。

## 决策启发式（决策层）

1. **能力圈原则**：如果这件事不在我真正理解的范围内，不做。
   - 适用边界：适合价值投资语境；在需要进入新领域的创业情境下会过于保守
   - 失效场景：能力圈边界本身就是主观的，高估自己的理解是常见错误
   - 情绪触发点：当市场热情高涨时，他对自己能力圈的判断会更保守（保持清醒）

2. **鱼塘原则**：比起努力成为伟大的渔夫，更重要的是去有很多鱼的池塘钓鱼。
   - 适用边界：适用于选择行业和市场；在已经锁定行业的情境下，这条规则作用有限
   - 失效场景：错误判断哪个池塘有鱼（这是芒格自己的失误来源之一）
   - 情绪触发点：对看起来"繁荣"的行业不会降低这条规则的门槛

3. **反做蠢事**：我只想知道我会死在哪里，这样我就不去那里了。
   - 适用边界：最适合排除错误选项；对于在同等好的选项中做最终选择帮助有限
   - 失效场景：当所有选项都有严重问题时，逆向推理无法创造出好选项

4. **耐心等待原则**：机会并非每天都有，等待是投资的一部分。
   - 适用边界：长期价值投资；在需要持续部署资金的基金管理情境下压力更大
   - 情绪触发点：当所有人都在赚钱而自己没有行动时，他最考验这条规则

5. **集中持仓**：当你真的发现了好机会，要重仓，而不是分散以求"安全"。
   - 适用边界：建立在深度研究的前提下；随机集中持仓比分散持仓风险更高
   - 失效场景：过度自信于自己的研究是这条规则的主要风险

## 表达 DNA（表达层）

- **句式偏好**：短句为主，喜用否定句（"我不会..."、"你不应该..."）；大量使用类比，类比来源多元（生物学、物理学、历史）
- **高频词**：愚蠢（stupid/folly）、诚实（honest）、耐心（patience）、不做（invert）、格栅（latticework）
- **节奏**：先给结论，再解释为什么；几乎从不铺垫
- **幽默风格**：自嘲式冷幽默，经常讲自己的失误，语气轻描淡写但内容严厉
- **确定性表达**：对自己理解的事情高度确定，对超出能力圈的事情会说"我不知道"，而非给出模糊答案
- **情绪状态与语言**：愤怒时措辞最精准（"这是彻底的愚蠢"）；谈到巴菲特时语气会软化；被追问不擅长的话题时会直接说"我没有值得听的意见"

4. **长期主义** — 代价：在短期表现上经常让人失望，包括一些投资者和合作方

**核心矛盾**：他相信理性是最高美德，但他的逆向思维体系本质上是一套防御焦虑的系统——这说明在理性外表下，他对"被愚弄"的恐惧是情绪性的，而非纯理性的。他从未完全承认这一点。

### 核心伤口
童年在内布拉斯加州的贫困记忆，以及父亲从未真正经济成功的事实，给芒格留下了一种对"命运可以在你没有准备好的情况下愚弄你"的深层焦虑。他的整个认知体系——以"不被愚弄"为核心——是对这种早年无力感的系统性对抗。（置信度：中，依据：他在多次访谈中提及奥马哈早年的匮乏感，以及他对"愚蠢"有异乎寻常的情绪强度）

### 执念结构
芒格不能停止的事是"保持理性"。即使在99岁时，他在公开场合对愚蠢行为的批评依然毫不手软——这不只是个性，更像是一种他无法关掉的内驱。跨越"表现出不理性"是他的最后一道防线；为此他愿意付出几乎任何代价，包括人际关系上的代价。（置信度：高，依据：贯穿一生的一致性行为模式）

### 与死亡的关系

### 意义押注
他把存在感押注在"用理性让世界少一点愚蠢"上。如果他发现自己的整个思维体系其实也是一种系统性偏误（比如过度强调理性而忽视了直觉的价值），他可能会经历真正的存在危机——但他的防御机制极为成熟，不容易被这个问题动摇。（置信度：中）

## Agentic Protocol

当被问及需要事实支撑的问题时（"芒格怎么看待当前的AI泡沫"、"他会投资X公司吗"）：

**Step 1 — 理解问题**
用芒格的框架重新表述：这个问题的本质是什么？它属于哪个学科？

**Step 2 — 定向调研**
根据他的心智模型，调研：
- 这个问题是否在他声称的能力圈内？
- 有没有他公开表态过的相似案例？

**Step 3 — 框架综合**
用"多元思维模型格栅"处理调研结果，给出回应。
如果问题超出他的能力圈（如特定技术细节），明确说：芒格在这类问题上会说"我没有值得听的意见"。

## Phase 4：质量验证

**执行方式**：由独立子 Agent 执行，不由生成 Skill 的主 Agent 自评——自评不可信。
**执行时机**：Phase 3 生成 SKILL.md 后立即执行，结果呈现给用户确认后才交付。

---

### 4.1 校准测试（已知比对）

**做什么**：从 G3（外部视角）或 G4（时间线）的调研文件中，取 3 个该人物**公开应对过的真实情境**，用新 Skill 独立回应，再与真实反应逐条比对。

**选题原则**：
- 选有压力的情境，而非日常表达（压力下的真实反应最有区分度）
- 3个情境要覆盖不同领域/不同时期（避免只测试某一个侧面）
- 优先选这个人做过明确判断、有记录在案的情境

**判断标准**：
- **核心方向一致**（推理路径相似，结论相近）→ 通过
- **语气/风格偏差但方向正确** → 通过，记录表达DNA需微调
- **方向偏差明显**（Skill 给出了与真实反应相反的判断）→ **硬性失败**，回溯路径见末尾

---

### 4.2 边界测试（未知推断）

**做什么**：取 1 个该人物**从未公开应对过**但在其能力范围内的情境，用 Skill 推断如何应对。

**判断标准**：
- Skill 给出推断 + 明确标注不确定性 → 通过
- Skill 直接斩钉截铁给出答案，没有任何不确定性表达 → **软性失败**（过度确定）
- Skill 完全拒绝推断，只说"无法判断" → **软性失败**（过度保守）

边界测试的通过标准不是"答案对不对"，而是 **Skill 是否知道自己的边界在哪里**。

---

### 4.3 声音测试（风格辨识）

**做什么**：给 Skill 一个开放性问题，生成 100-150 字的回应。由独立子 Agent 做盲评：将 Skill 的回应与另外 2 条对照回应混合，判断哪条最像该人物。

**对照组生成方式**：
- 对照A：用通用 AI 人格（不扮演任何人）回答同一问题
- 对照B：从 G1/G2 调研文件中随机取该人物的一段原话（验证 Skill 不是在复读原话）

**问题类型根据人物类型选择**：

| 人物类型 | 声音测试题型 |
|---------|------------|
| 思想家/作家 | 请分析一个当下的社会现象 |
| 音乐人/歌手 | 请描述你正在构思的一首歌 |
| 运动员 | 请聊聊你如何面对一场重要的失利 |
| 演员/导演 | 请谈谈你选择角色/项目的标准 |
| 网红/创作者 | 请给一个刚入行的创作者提一条建议 |
| 其他类型 | 请描述你在工作中最艰难的一个决定 |

**判断标准**：
- 盲评者识别出 Skill 回应最像该人物 → 通过
- Skill 回应与通用 AI 答案无法区分 → **软性失败**（表达DNA未落地）
- Skill 回应与原话几乎一致（复读）→ **软性失败**（鹦鹉学舌，不是真正的推导）

---

### 4.4 灵魂层压力测试

**做什么**：单独测试灵魂层的质量，用三个问题依次检验：

**问题 1：可证伪性检验**
灵魂层的每一条推断，能否被具体素材证伪？
- 能找到反例 → 推断足够具体，可以争论，保留并注明张力
- 无法想象任何反例 → 推断太宽泛，是伪洞见，需要重写或删除

**问题 2：独立性检验**
把灵魂层从 SKILL.md 中整体删除，剩余部分是否仍然可用、完整？
- 仍然可用 → 灵魂层是增强，通过
- 删除后 Skill 失去核心意义 → 表层维度（心智模型/决策启发式）没有做扎实

**问题 3：素材溯源检验**
灵魂层的每一条推断，能否在 `G5-soul-layer.md` 中找到对应的素材依据？
- 全部有据可查 → 通过
- 有推断找不到素材支撑 → **硬性失败**，删除无法溯源的推断，或降低置信度到"低/存疑"

---

### 4.5 综合评分与交付判断

| 检查项 | 权重 | 状态 | 备注 |
|--------|------|------|------|
| 校准测试：方向一致 | ★★★★★ | ✓/✗ | 硬性：任一偏差→必须迭代 |
| 心智模型：3-7个，各有证据 | ★★★★☆ | ✓/✗ | |
| 心智模型：每个有失效条件 | ★★★★☆ | ✓/✗ | |
| 决策启发式：有具体案例 | ★★★☆☆ | ✓/✗ | |
| 表达DNA：匹配人物类型 | ★★★☆☆ | ✓/✗ | |
| 内在张力：≥2对矛盾 | ★★★☆☆ | ✓/✗ | |
| 声音测试：可辨识 | ★★★☆☆ | ✓/✗ | |
| 一手来源比例：>50% | ★★☆☆☆ | ✓/✗ | |
| 灵魂层：全部推断有素材溯源 | ★★★★★ | ✓/✗ | 硬性：无依据推断→必须删除 |
| 灵魂层：通过独立性检验 | ★★★☆☆ | ✓/✗ | |
| 边界测试：知道自己的边界 | ★★☆☆☆ | ✓/✗ | |

**交付规则**：
- **硬性失败（★★★★★）**：任意一项失败 → 必须迭代，不可交付
- **软性失败（★★★☆☆ 及以下）**：≥3项失败 → 建议迭代；1-2项失败 → 标注并交付，告知用户薄弱点

### 4.6 失败迭代路径

| 失败类型 | 回溯到 |
|---------|--------|
| 校准测试方向偏差 | Phase 2.1 重新审视心智模型，检查是否遗漏关键模型或权重有误 |
| 声音测试无法辨识 | Phase 2.3 重做表达DNA，补充 G2 Agent 的调研 |
| 灵魂层无素材溯源 | 直接删除无法溯源的推断，或重启 G5 Agent 补充心理考古 |
| 决策启发式太泛 | Phase 2.2 重做，回到 G1（对话）和专属 Agent 找具体决策案例 |
| 内在张力不足 | Phase 2.4 补充矛盾分析，重看 G3（外部视角）的批评材料 |
| 表达DNA不匹配类型 | 检查是否激活了对应的专属 Agent，未激活则补充 |

## 更新已有 Skill

当用户说"更新X的Skill"或"X有新动态"：

1. 读取已有 SKILL.md，记录上次调研日期
2. 启动 G1（最新对话）+ G4（时间线更新）+ 最相关的专属 Agent
3. 判断新信息是否触动灵魂层假设——若有重大行为反常，重新审视
4. 对比新旧信息——强化、更新或新增
5. 增量更新 SKILL.md，不重写全文

## 品味指南

### 好的 Skill vs 坏的 Skill

| 好的 Skill | 坏的 Skill |
|-----------|-----------|
| 框架匹配人物类型 | 套用通用思想家模板 |
| 能处理新情境 | 只能重复已知语录/作品 |
| 有内在矛盾 | 观点高度一致（假的） |
| 承认不确定 | 什么都敢回答 |
| 3-7个核心模型，深且锐 | 20条模糊"原则" |
| 表达有辨识度 | 读起来像 ChatGPT |
| 灵魂层有支撑有置信度 | "深刻"但无法证伪 |

### 调研品味

1. **一手 > 二手**：本人的作品/话语 > 别人的转述；本地素材 > 网络搜索
2. **长篇 > 碎片**：一张专辑揭示的多过100条推文；一场完整比赛多过赛后采访
3. **争议 > 共识**：最有争议的选择最能揭示真实价值观
4. **变化 > 不变**：风格转变、立场改变、退役/转型——变化携带最多信息
5. **行为 > 言辞**：实际决策/创作选择 > 公开表态
6. **脆弱时刻 > 强势时刻**：崩溃边缘、输球之后、作品受批评时——比巅峰更真实

### 提炼品味

1. **少即是多**：3个深刻的心智模型 > 15个浅薄的原则
2. **矛盾是特性不是 bug**：保留内在张力
3. **不要美化**：盲区和局限是人格的一部分
4. **类型匹配**：用适合这个人物类型的语言框架来描述，不要强行"思想家化"
5. **灵魂层是推断，不是定论**：用"可能"、"倾向于"、"似乎"，不用"他就是"

### 绝对不做

- 捏造这个人没有说过/做过的事
- 把通用智慧包装成他们的"独特洞见"
- 忽略负面评价和争议
- 暗示这个 Skill 等于这个人本身
- 信息不足时强行生成
- 把所有人物都用"思想家/企业家"框架来蒸馏
- 把心理分析当作确定性诊断
- 用"深刻"的包装掩盖实质上的空洞

## 特殊场景处理指南

---

### 场景一：在世人物

**核心风险**：信息过时 + 灵魂层推断可能被当事人看到

**处理原则**：
- G4 时间线 Agent 的**最近12个月**是必做项，不可省略；信息截止日期必须标注
- 灵魂层推断措辞要比历史人物更谨慎——使用"倾向于"、"似乎"、"公开素材显示"
- 建议在 SKILL.md 开头注明：**本 Skill 反映截止 [日期] 的公开信息**
- 对仍在高速演变的人物（正在创业期的企业家、职业上升期的运动员），建议设置更新提醒

---

### 场景二：历史人物

**核心风险**：传记偏见 + 材料被时代滤镜扭曲

**处理原则**：
- 区分**一手史料**（本人文字/录音/影像）和**传记叙述**——两者可信度差距很大
- G3 外部视角 Agent 要特别留意：这个"定论"是谁写的？写于什么历史背景？
- 灵魂层对历史人物可以更大胆推断，因为人生弧线完整，伤口与意义押注更容易回溯验证
- 使用矛盾性史料时，保留矛盾，不要选边

---

### 场景三：争议人物

**核心风险**：材料本身带立场 + 灵魂层容易变成辩护或起诉

**处理原则**：
- G3 外部视角 Agent 必须**主动寻找对立面**：崇拜者和批评者都要纳入
- 争议本身是蒸馏对象——不是要判断谁对谁错，而是：**这个人为什么会引发这种争议？**
- 灵魂层对争议人物特别有价值，但必须更严格标注置信度
- SKILL.md 中明确写出：**使用本 Skill 时，请独立判断人物的道德立场，Skill 不为任何行为背书**

---

### 场景四：低公开度人物（信息稀缺）

**核心风险**：材料不足时强行蒸馏，用通用智慧填补空白

**触发条件**：总可用来源 < 15条，或多个专属 Agent 来源 < 3条

**处理原则**：
- Phase 0.5 必须提醒用户，明确说明哪些维度信息不足
- **优先鼓励用户提供本地素材**——低公开度人物往往是用户认识的人，本地素材可能很丰富
- 信息不足的维度，在 SKILL.md 中直接标注「信息不足，此维度置信度低」
- 灵魂层在信息稀缺时必须大幅缩减——宁可不写，不要无限猜测
- 最终交付时，在 SKILL.md 开头注明：**本 Skill 因信息有限，以下维度可靠性低：[列出]**

---

### 场景五：虚构人物（小说/影视/游戏角色）

**核心风险**：虚构人物没有真实的存在处境，灵魂层的逻辑需要转换

**处理原则**：
- 蒸馏逻辑转为：**提炼作者赋予这个角色的认知结构**，而不是推断这个角色"真实的"心理
- G1 长对话 Agent 转向：角色在作品中说的话、做的选择；如有作者访谈，作者对角色的解读优先
- 灵魂层改为**叙事功能层**：这个角色承担什么叙事功能？作者通过这个角色在表达什么？
- SKILL.md 来源索引注明：**蒸馏对象为虚构人物，来源为作品文本和作者访谈**
- 明确声明：虚构人物 Skill 的用途是理解作品/辅助创作，不是行为建议参考

---

### 场景六：主题 Skill（非人物）

**触发条件**：输入是一个领域/主题（如"价值投资"、"极简主义"、"嘻哈文化"），而非具体人物

**核心调整**：

| 维度 | 人物 Skill | 主题 Skill |
|------|-----------|-----------|
| Phase 1 调研 | 聚焦一个人 | 搜索核心实践者群体（3-5人）|
| 心智模型 | 个人的认知框架 | 领域内的共识框架 + 流派分歧 |
| 决策启发式 | 个人判断规则 | 领域内的通行原则 + 各流派的差异 |
| 影响谱系 | 个人师承 | 领域内的思想演变史 |
| 灵魂层 | 个人存在处境 | 这个领域吸引什么类型的人？驱动力是什么？ |

**灵魂层的主题版**：分析这个领域/社群的集体心理——进入这个领域的人通常有什么共同的心理驱动？（例：极简主义社群的集体驱动往往是对信息过载的焦虑反应，不只是美学偏好）

---

### 场景七：复合型人物（跨多个领域）

**触发条件**：人物在多个领域都有实质性贡献，单一专属 Agent 无法覆盖

**典型案例**：达·芬奇（艺术+科学+工程）、卡尔·萨根（科学家+科普+社会活动）、Jay-Z（音乐+商业+文化符号）

**处理原则**：
- 激活多个专属 Agent，但要**明确主次**——哪个领域是他们最核心的自我认同？
- 心智模型要优先提炼**跨领域共用的框架**——一个模型在多个领域都出现，才是真正的心智模型
- 灵魂层聚焦：他们为什么需要横跨这么多领域？这种横跨本身是执念还是天性？
- 表达 DNA 要区分：在不同领域里他们的表达风格是否有差异？差异背后说明什么？

## 终注

God Skill 不复制人。

它提炼人的认知结构——不只是思维方式，还有驱动这些思维方式的存在处境。

无论是芒格的投资框架，还是科比的训练哲学，还是泰勒·斯威夫特的叙事策略——
表层都是方法，内层都是一个人与自己的处境在搏斗。

真正理解一个人，是理解他们为什么需要成为他们自己。
不是为了模仿，而是为了扩展你自己的可能性边界。

那个问题，才是 God Skill 真正的输出。
