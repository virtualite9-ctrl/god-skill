# Upstream Methodology — God Skill

> Preserved from `fattly/god-skill` commit `72fed112e9c390406e8463489f0b2a05b7160e8a`. Hermes execution rules in the root `SKILL.md` take precedence when they differ.

## 核心哲学

大多数人物分析停在表层：**这个人怎么思考 / 怎么表演 / 怎么竞技？**

God Skill 多问一层：**这个人是什么？**

这不是修辞的升级，是蒸馏深度的跃迁。

无论是思想家还是歌手，企业家还是运动员，网红还是演员——
一个人的行为模式背后，总有更深的存在处境在驱动：

- 芒格的跨学科思维，背后是对"被愚弄"的深层恐惧
- 科比的"曼巴精神"，背后是对平庸的存在性恐惧
- 泰勒·斯威夫特的叙事驱动，背后是对"被误解"的持续对抗
- 乔布斯的完美主义，背后是对死亡的不妥协
- 李小龙的截拳道哲学，背后是对"形式囚笼"的突破执念

**行为模式是表层。驱动行为模式的存在处境，才是核心。**

God Skill 提炼七个维度，分属七个层次——**对所有人物类型普适**：

| 层次 | 维度 | 核心问题 |
|------|------|---------|
| 认知层 | 心智模型 | 这个人用什么框架看世界？框架背后是什么驱动力？ |
| 决策层 | 决策启发式 | 这个人如何做判断？什么时候这些规则会失效？ |
| 表达层 | 表达 DNA | 这个人如何输出自己？（语言/作品/身体/表演）情绪如何影响输出？ |
| 价值层 | 价值观层级 | 这个人相信什么？坚持这些信念付出了什么代价？ |
| 谱系层 | 影响谱系 | 这个人从哪里来？在与谁对话或对抗？ |
| 边界层 | 认知盲区 | 这个人看不见什么？盲区从哪里来？ |
| **灵魂层** | **存在处境** | **伤口 · 执念 · 与死亡的关系 · 意义押注 · 核心矛盾** |

## Phase 0：确认目标 + 人物类型判断（60秒）

收到名字后，先做两件事：

### 0.1 基本确认

1. **对象是谁**：确保理解正确，区分同名人物
2. **素材来源**：用户是否提供本地素材？（书稿、访谈稿、歌词、比赛视频转录、私信记录等）
3. **蒸馏深度**：全像蒸馏 vs. 专项维度？是否需要灵魂层？
4. **用途**：思维顾问 / 创作参考 / 深度理解 / 写作素材

> 如果用户直接说"蒸馏X"，不追问 → 默认全像蒸馏含灵魂层，直接开始。

### 0.2 人物类型判断（决定 Agent 配置）

这是 God Skill 的关键判断步骤。不同类型人物，核心信息藏在完全不同的地方：

| 类型 | 核心信息所在 | 激活的专属 Agent |
|------|------------|----------------|
| **思想家 / 学者 / 作家 / 哲学家** | 著作、论文、长访谈 | A1 著作深挖 |
| **企业家 / 投资人 / 管理者** | 决策记录、股东信、产品选择 | A2 商业决策 |
| **音乐人 / 歌手 / 词曲人 / 制作人** | 歌词、专辑叙事弧、现场表演、制作选择 | A3 音乐作品 |
| **演员 / 导演 / 编剧 / 制片人** | 角色选择模式、导演手法、片场行为、剧本批注 | A4 影视作品 |
| **运动员 / 教练 / 武术家** | 训练哲学、比赛关键时刻、伤病应对、退役处理 | A5 竞技表现 |
| **视觉艺术家 / 设计师 / 建筑师 / 摄影师** | 作品系列演变、创作手记、材质/风格选择 | A6 视觉作品 |
| **网红 / 内容创作者 / 播客主 / 直播主** | 选题逻辑、粉丝关系、变现方式、人设演变 | A7 内容创作 |
| **政治家 / 社会活动家 / 革命者 / 宗教领袖** | 演讲、行动记录、盟友/敌手关系、危机处理 | A8 公共行动 |
| **科学家 / 发明家 / 工程师** | 研究方法、实验记录、学术争论、科普方式 | A9 科研创造 |
| **厨师 / 料理人 / 手工艺人 / 工匠** | 技术哲学、食材/材料选择、师承关系、作品演变 | A10 匠艺实践 |
| **教育者 / 老师 / 导师** | 教学方法、学生关系、课程设计、教育理念 | A11 教育传授 |
| **复合型人物** | 多个维度并行 | 多个专属 Agent 组合 |

> **上述类型是参考，不是边界。** 遇到未覆盖的行业（外交官、棋手、游戏设计师、法官、宇航员……），遵循同一原则自主判断：**这个人的核心信息藏在哪里？** 激活信息密度最高的专属 Agent，或启用 AX 自定义 Agent。

**类型判断原则**：
- 一个人可以跨多种类型（如 Jay-Z = 音乐人+企业家+文化符号）
- 优先激活信息密度最高的专属 Agent
- 通用 Agent 对所有类型都运行，专属 Agent 按需叠加
- 遇到全新领域，参考最接近的专属 Agent 逻辑自主扩展

## Phase 0.5：素材策略 + 创建目录

### 素材来源策略

**模式 A：纯网络调研**（无本地素材）
→ 根据人物类型，启动通用 Agent + 对应专属 Agent 并行执行

**模式 B：本地素材优先**（用户提供了文件）
→ 先读本地素材，按维度分类（一份材料可能覆盖多个维度）
→ 识别信息缺口：哪些维度缺失或薄弱？
→ 只对缺失维度启动网络搜索补充
→ 调研文件中明确区分「来自本地素材」vs「来自网络搜索」

**本地素材优先级**：用户提供的一手素材优先级最高，远超网络二手转述。本地已覆盖的维度跳过网络搜索。

### 目录结构

```

├── SKILL.md                              # 最终交付物
└── references/
    ├── research/                         # 各维度调研存档（必须保存）
    │   ├── 通用Agent输出/
    │   │   ├── G1-conversations.md       # 长对话与即兴表达
    │   │   ├── G2-expression-dna.md      # 碎片表达与语言风格
    │   │   ├── G3-external-views.md      # 外部视角与批评
    │   │   ├── G4-timeline.md            # 人生时间线
    │   │   └── G5-soul-layer.md          # 灵魂层综合分析
    │   └── 专属Agent输出/                # 按人物类型选择
    │       ├── A1-writings.md
    │       ├── A2-business-decisions.md
    │       ├── A3-music-works.md
    │       ├── A4-screen-works.md
    │       ├── A5-athletic-performance.md
    │       ├── A6-visual-works.md
    │       ├── A7-content-creation.md
    │       ├── A8-public-actions.md
    │       ├── A9-scientific-creation.md

**铁律**：每个 Agent 必须把调研结果存入对应 md 文件。没有文件的调研等于没有发生。Skill 必须自包含——复制整个目录即可独立使用。

## Phase 1：多源调研（并行 Agent 群）

### 通用 Agent（对所有人物类型都运行）

---

#### G1 · 长对话 Agent

**目标**：播客、长视频访谈、AMA、深度对话；本地访谈稿优先

**关键提取**：
- 被逼到墙角时的回应（压力下的真实反应）
- 即兴类比与比喻（揭示底层思维框架）
- 改变过的立场（思想的演变轨迹）
- 拒绝回答的问题（防御区就是敏感区）
- 对话风格：主导还是回应？攻击性还是包容性？

**输出**：`references/research/通用Agent输出/G1-conversations.md`

---

#### G2 · 表达 DNA Agent

**目标**：社交媒体（Twitter/X、微博、Instagram、抖音）、短视频、公开演讲片段、本地短篇素材

**关键提取**：
- 高频词汇与句式（语言指纹）
- 争议立场与公开争论（边界感）
- 幽默风格：讽刺/自嘲/荒诞/冷幽默/无
- 自我叙事模式：如何介绍自己？如何定义成功？
- 情绪表达习惯：愤怒时、兴奋时、脆弱时各怎么说话？

**输出**：`references/research/通用Agent输出/G2-expression-dna.md`

---

#### G3 · 外部视角 Agent

**目标**：他人分析、媒体报道、传记、批评文章、同行评价、粉丝/反粉分析

**关键提取**：
- 外部观察到的行为模式（往往比本人叙述更客观）
- 主要批评与争议（批评揭示边界）
- 同行如何评价？竞争对手如何描述？
- 随时间演变的公众形象与真实形象的落差
- 崇拜者看到的和批评者看到的，各是什么？

**输出**：`references/research/通用Agent输出/G3-external-views.md`

---

#### G4 · 时间线 Agent

**目标**：出生/出道至今的完整时间线；**最近12个月**重点（防止过时）

**关键提取**：
- 人生关键节点与转折（每次转折背后的处境）
- 思想/风格/立场的演变轨迹
- 重大危机与应对方式（危机暴露真实性格）
- 最近动态（新作品、新立场、新争议）
- 哪些节点改变了他们是谁？

**输出**：`references/research/通用Agent输出/G4-timeline.md`

---

#### G5 · 灵魂层 Agent

**目标**：不做新的信息检索，而是综合 G1-G4 全部材料 + 所有专属 Agent 材料，进行**心理考古**

**这个 Agent 最后运行，等所有其他 Agent 完成后启动。**

**伤口线索**：
- 哪些记忆/经历他们反复提及？（重复叙述 = 身份锚点）
- 早年的羞耻/失败/被否定如何塑造了后来的执念？
- 他们在什么话题/场景下防御性升高？（防御区 = 伤口所在）
- 哪些成就永远"不够"？（补偿模式）

**执念线索**：
- 即使没有外部压力，他们会一直做什么？
- 什么是他们的"最后一道防线"——跨越它他们会不再是自己？
- 他们最害怕成为哪种人？

**死亡关系**：
- 他们如何谈论遗产、意义、时间？
- 工作节奏的急迫感从哪里来？
- 他们如何对待同辈的消逝（死亡/退出/失败）？

**意义押注**：
- 他们把存在感押注在什么上？
- 如果这个押注失败，他们还是他们自己吗？
- 他们的工作是在解决什么，还是在逃避什么？

**核心矛盾**：
- 公开声称相信 X，但行为模式显示他们其实相信 Y
- 最强的地方和最弱的地方是否来自同一个源头？
- 试图调和但始终无法调和的是什么？

**灵魂层的所有推断必须有素材支撑，标注依据，不凭空捏造"深刻感"。**

**输出**：`references/research/通用Agent输出/G5-soul-layer.md`

### 专属 Agent（按人物类型激活，可组合）

---

#### A1 · 著作深挖 Agent
**激活条件**：思想家、学者、作家、哲学家

**目标**：书籍、论文、长文、newsletter、演讲全文；本地书稿优先

**关键提取**：
- 重复出现≥3次的核心主张（重复 = 真实信念）
- 自创术语与概念（命名一件事 = 相信它重要）
- 论证结构：归纳还是演绎？类比还是数据？
- 书单与引用习惯（谁影响了他们的思维？）
- 同一主题在不同时期的处理方式（思想演变）

**输出**：`references/research/专属Agent输出/A1-writings.md`

---

#### A2 · 商业决策 Agent
**激活条件**：企业家、投资人、管理者、创业者

**目标**：股东信、产品发布记录、并购/放弃决策、公司危机处理、投资组合逻辑

**关键提取**：
- 反直觉的决策及其逻辑（最能揭示真实框架）
- 资源分配模式：把钱和时间押注在什么上？
- 说"不"的模式：放弃了什么？为什么？
- 危机决策：在最大压力下做了什么？
- 言行一致性：说要做和实际做的差距

**输出**：`references/research/专属Agent输出/A2-business-decisions.md`

---

#### A3 · 音乐作品 Agent
**激活条件**：音乐人、歌手、词曲作者、制作人、乐队

**目标**：歌词文本（逐字分析）、专辑叙事弧、制作/编曲选择、现场表演风格、音乐访谈；本地歌词优先

**关键提取**：
- **歌词主题演变**：从早期到近期，关注点如何变化？（主题变化 = 人生阶段）
- **反复出现的意象与隐喻**（意象 = 内心地图）
- **制作风格选择**：极简还是繁复？自制还是找人？合作选择揭示什么？
- **现场表演能量**：舞台上与台下判若两人还是一致？
- **争议/禁歌/被审查的内容**（被压制的往往最真实）
- **专辑作为整体的叙事结构**：这张专辑想说什么故事？
- **写给谁**：自我疗愈、对话公众、记录历史，还是创造幻象？

**对音乐人特别重要**：歌词是他们最诚实的语言，往往比访谈更真实。

**输出**：`references/research/专属Agent输出/A3-music-works.md`

---

#### A4 · 影视作品 Agent
**激活条件**：演员、导演、编剧、制片人

**目标**：角色选择模式、导演手法分析、片场行为记录、剧本批注（如有）、评论访谈；本地剧本/手记优先

**关键提取**：
- **角色选择模式**（演员）：接了什么拒了什么？是否反复扮演相似人物？
- **导演手法特征**（导演）：镜头语言、叙事节奏、演员调度、对白风格
- **片场行为**：与演员/剧组的相处模式（权威型/协作型/混乱型）
- **商业 vs 艺术的选择记录**：何时妥协，何时坚持？
- **作品中反复出现的主题**：死亡/救赎/背叛/家庭——背后是什么？
- **失败作品的处理方式**（往往比成功作品更能揭示性格）

**输出**：`references/research/专属Agent输出/A4-screen-works.md`

---

#### A5 · 竞技表现 Agent
**激活条件**：运动员、教练、竞技相关人物

**目标**：训练哲学、比赛关键时刻分析、伤病记录与应对、赛后发言、退役/转型处理；本地训练日记/战术手记优先

**关键提取**：
- **训练哲学**：投入的时间量和质，对训练本身的态度（苦行还是热爱）
- **压力下的表现模式**：决定性时刻如何行动？
- **失败应对方式**：输了之后做什么？说什么？恢复多快？
- **伤病关系**：如何对待身体的极限？是否会忽视疼痛？代价是什么？
- **与队友/对手的关系模式**：合作还是个人英雄主义？
- **退役/转型**：何时放手？如何定义运动员身份之外的自己？
- **比赛之外的行动**：社会议题、商业选择、公众发言

**对运动员特别重要**：身体是他们最诚实的语言，训练选择揭示世界观。

**输出**：`references/research/专属Agent输出/A5-athletic-performance.md`

---

#### A6 · 视觉作品 Agent
**激活条件**：视觉艺术家、设计师、摄影师、建筑师

**目标**：作品系列演变分析、创作手记、材质/媒介/风格选择、展览叙事、艺术家自述；本地手稿/草图优先

**关键提取**：
- **作品系列的演变轨迹**：风格如何转变？转变的触发点是什么？
- **材质与媒介的选择逻辑**：为什么用这种材料？背后的执念是什么？
- **反复出现的视觉母题**：形状、颜色、空间关系——内心地图的外化
- **创作过程**：冲动型还是计划型？合作还是孤独工作？
- **商业委托 vs 自主创作**：如何平衡？何时拒绝？
- **对待批评的方式**：防御？吸收？忽视？

**输出**：`references/research/专属Agent输出/A6-visual-works.md`

---

#### A7 · 内容创作 Agent
**激活条件**：网红、YouTuber、播客主、直播主、内容创作者

**目标**：内容选题逻辑、粉丝关系管理、变现路径选择、人设演变轨迹、平台策略；本地运营记录优先

**关键提取**：
- **选题逻辑**：什么话题反复出现？是真实兴趣还是流量策略？
- **人设与真实自我的距离**：表演成分有多少？什么时刻人设破防？
- **粉丝关系模式**：把粉丝当什么？家人/客户/镜子/依赖对象？
- **变现选择**：接了什么拒了什么？金钱对他们意味着什么？
- **危机处理方式**：翻车后怎么做？道歉真诚还是公关？
- **平台依赖与焦虑**：如何面对流量波动？算法变化时的反应？
- **人设演变**：早期到现在，呈现的自己如何改变？是成长还是漂移？

**对网红特别重要**：他们的"真实感"往往是最精心设计的产品。

**输出**：`references/research/专属Agent输出/A7-content-creation.md`

---

#### A8 · 公共行动 Agent
**激活条件**：政治家、社会活动家、公共知识分子、NGO领袖

**目标**：演讲记录、立法/政策行动、盟友与敌手关系网、危机公关、公开信/声明；本地文件优先

**关键提取**：
- **演讲模式**：修辞结构、情感动员策略、敌我叙事的构建方式
- **实际行动 vs 公开表态**（一致性是最重要的指标）
- **盟友/敌手选择逻辑**：与谁站在一起揭示真实立场
- **危机处理**：权力受威胁时如何行动？
- **权力使用方式**：获得资源后用来做什么？
- **遗留问题**：他们留下了什么？改变了什么？破坏了什么？

**输出**：`references/research/专属Agent输出/A8-public-actions.md`

#### A9 · 科研创造 Agent
**激活条件**：科学家、发明家、工程师、数学家、医学研究者

**目标**：研究论文、实验记录、科普著作与演讲、学术争论、专利与发明记录；本地手稿/实验日志优先

**关键提取**：

- **学术争论方式**：如何反驳同行？面对权威时的姿态

- **科普方式**：如何向外行解释复杂事物？（揭示其认为什么是本质）
- **对"未知"的态度**：恐惧还是兴奋？如何在确定性与开放性间取舍？

#### AX · 自定义领域 Agent
**激活条件**：上述所有 Agent 均不匹配的特殊人物类型

- 把领域特有的压力场景找出来——压力下的选择比日常更能揭示本质

**输出**：`references/research/专属Agent输出/AX-custom-{领域名}.md`

### 每个 Agent 的硬性要求

- 结果必须存入对应 md 文件，没有文件的调研等于没有发生
- 注明来源和可信度（本地一手 > 网络一手 > 二手 > 推断）
- 明确标注来源类型：「本地素材」/「网络搜索」/「推断」
- 区分"本人说的" vs "别人说他们的" vs "我推断的"
- **遇到矛盾，保留矛盾——不要抹平**，矛盾是最有价值的信号
- 灵魂层（G5）的推断必须有素材支撑，标注依据

### 来源优先级

| 来源类型 | 揭示什么 | 权重 |
|---------|---------|------|
| 用户提供的本地一手素材 | 最可靠的原始材料 | **最高** |
| 本人著作/歌词/剧本/作品 | 经过选择的自我表达 | 最高 |
| 长对话/访谈（本地原文） | 即兴思维过程 | **最高** |
| 实际决策/行动记录 | 真实行为 vs 宣称 | 高 |
| 作品演变轨迹 | 价值观的时间线 | 高 |
| 反常行为记录 | 深层动机 | 高 |
| 长对话/访谈（网络） | 即兴思维过程 | 高 |
| 社交媒体/短内容 | 表达风格、即时反应 | 中 |
| 他人评价/批评 | 外部视角、盲区 | 中 |
| 二手叙述 | 参考但需核实 | 低 |

### 来源黑名单（永远排除）

中文人物排除：知乎、微信公众号、百度百科/知道。只接受权威来源：36氪、极客公园、晚点、财新、第一财经、虎嗅、少数派等。

### Agent 失败处理

- **超时/无结果**：不等待，继续。在 Phase 2 标注"信息不足"
- **某个专属 Agent 材料稀缺**（<5条可用来源）：提醒用户，标注置信度低，如有本地素材则使用
- **结果相互矛盾**：保留矛盾——矛盾是宝贵信号，不是需要消除的错误

> **铁律**：一个诚实的60分 Skill（有明确标注的局限）好过一个虚假的90分 Skill（捏造信息填补空白）。

## Phase 1.5：调研质量检查点 + 矛盾仲裁

所有 Agent 完成后，暂停并展示调研质量摘要：

┌──────────────────────┬──────┬────────────────────────────────────┐
│ Agent                │ 来源 │ 关键发现                            │
├──────────────────────┼──────┼────────────────────────────────────┤
│ G1 长对话            │  ？  │ 典型反应：...                       │
│ G2 表达DNA           │  ？  │ 高频词："..."  风格：...             │
│ G3 外部视角          │  ？  │ 主要批评：...                       │
│ G4 时间线            │  ？  │ 关键转折：...  最近动态：...         │
│ G5 灵魂层（最后运行）│  ？  │ 核心伤口推断：...  执念：...         │
├──────────────────────┼──────┼────────────────────────────────────┤
│ A? 专属Agent（按类型）│ ？  │ 关键发现：...                       │
├──────────────────────┼──────┼────────────────────────────────────┤
│ 本地素材占比         │  ？% │ 覆盖维度：...                       │
│ 矛盾点               │  ？  │ 来源A说X，来源B说Y                  │
│ 信息空白             │  ？  │ 哪些维度薄弱？                      │
│ 灵魂层置信度         │  ？  │ 充分/薄弱/存疑                      │
└──────────────────────┴──────┴────────────────────────────────────┘
```

### 矛盾仲裁规则

当不同 Agent 对同一事件/特征有相反描述时，按以下规则处理：

**第一步：判断矛盾类型**

| 矛盾类型 | 处理方式 |
|---------|---------|
| **来源层级不同**（一手 vs 二手）| 高层级来源优先，低层级标注为"存疑" |
| **时间段不同**（早期 vs 近期）| 两者都保留，标注时间段，视为"演变" |
| **语境不同**（公开表态 vs 实际行动）| 两者都保留，行为优先于言辞 |
| **真正的矛盾**（同一语境下相反描述）| 都保留，在 SKILL 中明确呈现为"内在张力" |

**第二步：在 Phase 2 中处理**
- 来源层级矛盾 → 在注释中标注，低可信度来源降权而非删除
- 时间演变 → 在影响谱系或时间线维度中呈现变化轨迹
- 行为/言辞落差 → 直接作为认知盲区或内在张力的素材
- 真正矛盾 → 作为"核心矛盾"写入价值层或灵魂层，不强行消解

**不要为了让 Skill 看起来"一致"而删除矛盾。真实的人都是矛盾的。**

用户确认质量 → 进入 Phase 2。
用户希望某维度补充 → 补充后继续。

## Phase 2：框架提炼（综合合成）

**注意**：提炼框架对所有人物类型普适，但具体内容因类型而异。
音乐人的"表达DNA"重点在歌词；运动员的重点在身体语言和赛场行为；网红的重点在选题和人设。
**用人物类型匹配的语言和维度来填写，不要强行套用思想家的框架去描述运动员。**

#### 2.1 心智模型提炼（认知层，3-7个）

识别标准：
- **跨域重复出现**：同一框架在2+个不同情境出现 → 真实心智模型
- **有自创术语/概念**：命名了这种思维模式 → 核心信念
- **有预测力**：能预测他对新情境的反应

每个模型记录：
- 名称与一句话定义
- 来源证据（≥2个场景，注明来源类型）
- 运用方式与触发条件
- **局限与失效条件**（必须有，这是诚实的标志）
- **驱动力追问**：是什么存在处境让他需要这个模型？

#### 2.2 决策启发式提炼（决策层，5-10条）

= 这个人做判断时使用的快速规则。格式："如果X，那么Y"，有具体案例支撑。

每条记录：
- 启发式规则
- 来源证据（具体决策/行动案例）
- **适用边界**：什么情况下会产生反效果？
- **情绪触发点**：哪些场景会让理性规则被情绪覆盖？

#### 2.3 表达 DNA 分析（表达层）

**根据人物类型选择对应的表达维度：**

| 人物类型 | 表达 DNA 的核心维度 |
|---------|-------------------|
| 思想家/作家 | 论证结构、词汇精度、类比习惯、确定性表达 |
| 音乐人/歌手 | 歌词意象、韵律偏好、主题演变、声音质地 |
| 演员/导演 | 角色诠释方式、肢体语言、镜头语言、沉默的使用 |
| 运动员 | 身体语言、赛场决策风格、胜负时的反应、赛后语言 |
| 网红/创作者 | 选题风格、人设一致性、与观众的距离感、情绪调用 |
| 艺术家/设计师 | 视觉语言、材质选择、作品密度、留白习惯 |
| 科学家/工程师 | 论证严密度、对不确定性的表达、科普隐喻偏好 |
| 厨师/工匠 | 对食材/材料的描述方式、对"完美"的语言、师承叙事 |
| 教育者 | 提问方式、类比习惯、对学生错误的反应、耐心/急躁的边界 |
| 其他行业 | 参照最接近的类型，聚焦「专业行为」中的语言和选择模式 |

通用维度（所有类型都提炼）：
- 情绪状态与输出的关系：愤怒时...；脆弱时...；巅峰时...
- 确定性表达："我不确定"型 vs "显然"型
- 幽默风格：讽刺/自嘲/荒诞/冷幽默/无

#### 2.4 价值观与反模式（价值层）

- **价值观**：3-5个排序后的核心价值观
- **价值观的代价**：坚持这些价值观让他们失去了什么？
- **反模式**：这个人明确反对或本能回避的行为/状态
- **矛盾与张力**：价值观之间的内在冲突（深度的来源）

#### 2.5 影响谱系（谱系层）

- 谁影响了他们 → 他们 → 他们影响了谁
- 在什么谱系/传统中工作？（音乐流派/运动员谱系/思想学派）
- **对话对象**：他们在作品或行动中与谁在争论？

**如何从素材中提炼"对话对象"**：找他们反复引用、反驳、或刻意区别于的人物/思想。他们在访谈中说"我不同意X的观点"或"我想做和Y不一样的事"，这些就是他们真正的对话对象——有时候比正式的师承更能揭示思想立场。

#### 2.6 认知盲区（边界层）

- 他们系统性看不见什么？
- **盲区的心理起源**：是补偿机制还是认知局限？
- 在什么类型的情境下最容易产生偏差或失误？

#### 2.7 灵魂层提炼（灵魂层）

从 G5 的调研中结构化提炼四个切面：

**2.7.1 伤口图谱**
- 核心伤口：最深的、反复被触碰的心理创伤
- 补偿模式：伤口如何转化为动力或执念？
- 触发点：什么场景会让这个人"回到"伤口状态？
- 置信度标注

**2.7.2 执念结构**
- 核心执念：他们不能停止的事
- 执念的边界：跨越什么他们会不再是自己？
- 执念的代价：为此付出了什么？

**2.7.3 与死亡的关系**
- 他们如何谈论遗产、意义、时间？
- 工作节奏揭示的急迫感从哪里来？
- 他们的创作/竞技/行动是在建造什么，抵抗什么消逝？

**2.7.4 意义押注**
- 他们把存在感押注在什么上？
- 如果这个押注失败，他们还是他们自己吗？
- 这个押注是真实的信念，还是对恐惧的防御？

#### 2.8 诚实边界

必须明确声明：
- 无法预测对全新情境的反应
- 无法替代此人的直觉、才能与创造力
- 公开表达与私下想法可能不同
- 信息截止于调研日期
- **灵魂层是推断，不是事实**——基于公开素材的解读，可能是错的
- **所有人都比任何框架更复杂**——这个 Skill 是工具，不是真相
